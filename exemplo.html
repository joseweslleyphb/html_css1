HTML

A linguagem HTML é a liguagem de marcação padrão para as páginas da web

Com Html é possivel criar o seu própio site

Tag Descrição Exemplo
<p> A tag P define um novo parágrafo <p>Isto é um parágrafo.</p>
<a> A tag A define um novo link <a href="url">Texto do link.</a>
<h1> A tag h1 define um novo título com importância 1 <h1>Texto do link.</h1>
<h2> A tag h2 define um novo título com importância 2 <h2>Texto do link.</h2>
<h3> A tag h3 define um novo título com importância 3 <h3>Texto do link.</h3>

CSS

CSS é a linguagem usada para estilizar um documento HTML.

O CSS descreve como os elementos HTML deverão ser exibidos.

Propriedade Descrição Exemplo
background-color A propiedade backgroud-color especifica a cor de fundo de um p { background-color:blue; }
background-image A propiedade backgroud-image especifica uma imagem para utilizar body { background-image: url("caminho_imagem.jpg") }
color A propiedade color especifica a cor da fonte dos elementos. h1 { color: red; }
text-align A propiedade text-align especifica o aliamento do texto p { text-align: center; }
font-family A propiedade font-family especifica qual a fonte será utilizada. body { font-family: verdana; }
